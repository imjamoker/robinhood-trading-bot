# Robinhood Trading Bot - Safe $50 Branch

## Purpose

This branch (`safe-50-dollar-bot`) is a learning and testing environment for automated trading.

Goals:

* Learn Python, Git, GitHub, and Linux deployment
* Learn how algorithmic trading systems work
* Gain experience with Robinhood Agentic Trading and MCP integrations
* Limit financial risk while learning

## Account Rules

Current safety limits:

* Total account size: $50
* Cash reserve: $10
* Tradeable capital: $40
* Maximum open positions: 2
* Maximum position size: $15
* Maximum buys per run: 2

## Important Notes

* Do not commit credentials or tokens to GitHub.
* Store Robinhood credentials only in local environment variables.
* Test all changes before enabling live trading.
* This project is intended for education and experimentation.

## Future Plans

* Deploy to Linux server
* Improve risk management
* Add better logging and reporting
* Evaluate trading performance over time
